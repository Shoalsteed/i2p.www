{{ _('Part Two: Download and Install I2P from a .jar file') }}

  1. {% trans -%}Download I2P for Unix from{%- endtrans %} https://geti2p.net.
  2. {% trans -%} Because I2P is being installed from a .jar file, it cannot be signed by an Apple certificate. You will need to allow it special permission to install. Even though the installer is unsigned, the updates are signed end-to-end by I2P. {%- endtrans %}
  3. {% trans -%}Select a language you are familiar with.{%- endtrans %}
  4. {% trans -%} Now the installer is ready to start. Click next to advance. {%- endtrans %}
  5. {% trans -%} Accept the license. I2P is Free Software, mostly in the public domain with limited use of GPL2, Creative Commons, and other Free and Open-Source Licenses. {%- endtrans %}
  6. {% trans -%} Install the I2P router and base config. It is recommended that you keep the install directory the default. {%- endtrans %}
  7. {% trans -%} I2P is now installed! The remaining installer pages explain some aspects of running I2P on OSX. {%- endtrans %}

{{ _('Part Three: Configure I2P App') }}

  1. {% trans -%} For convenience, you may want to create a shortcut to launch the I2P router. Find the "i2p" directory in the "Applications" directory using Finder. {%- endtrans %}
  2. {% trans -%}Open the folder and find the Start Router Icon shown.{%- endtrans %}
  3. {% trans -%}Click the icon to start the I2P router - it will show up in your dock as shown and you can choose too keep it there.{%- endtrans %}

{{ _('Part Four: Configure I2P Bandwidth') }}

  1. {% trans -%} When you visit the I2P router console for the first time, it will automatically direct you to the configuration wizard. Start by selecting a language for the I2P interface. {%- endtrans %}
  2. {% trans -%}Next, pick either a dark or light theme.{%- endtrans %}
  3. {% trans -%} The next step is the bandwidth test. The bandwidth test takes a minute to run completely. During the bandwidth test, we'll need to connect to the external M-Lab Service, which makes a direct connection to a remote server(Operated by Measurement Lab) to measure your internet speed. {%- endtrans %}
  4. {% trans -%} Confirm the bandwidth measurement and adjust your share percentage. {%- endtrans %}
  5. {% trans -%}Confirm your bandwidth settings and adjust how much of your bandwidth you wish to share.{%- endtrans %}
  6. {% trans -%}You're finished! I2P is now configured.{%- endtrans %}

{% trans -%}If you want to re-run the welcome wizard after completing it, you can visit the page on{%- endtrans %} your router console.
